# ニゲロにゃんにゃん

## ゲームの内容
オレンジ色のネコと青色のネコを同時に操作して、上から降ってくる矢印から逃げるゲーム。
左上と右上に円型のHPゲージがあるので、それが0にならないように注意する。

![画面イメージ](docs/images/game_image01.png)

## 操作説明
- オレンジのネコはAとDで左右に移動。青のネコは←と→で左右に移動。
